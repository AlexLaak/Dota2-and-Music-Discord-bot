# discoBotTesting

Bot testing

Pre req: Python v3.*
For sound playback:
```
add ffmpeg.exe to project folder
```
Dependencies:
```
py -3 -m pip install discord
py -3 -m pip install beautifulsoup4
py -3 -m pip install PyNaCl
py -3 -m pip install youtube-dl

```
    
Running on bash:
```
cd <project folder>
python Discord.py
```
