# discoBotTesting

Bot testing

Pre req: Python v3.*

For sound playback:
```
add ffmpeg.exe to project folder
```
For building and running on bash:
```
py -3 -m pip install discord
py -3 -m pip install beautifulsoup4
py -3 -m pip install PyNaCl
cd ~
touch .bashrc
sudo .bashrc
Add this line: alias python='winpty python.exe'
```
    
Running on bash:
```
python Discord.py
```
